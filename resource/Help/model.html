Load EMF (Eclipse Modelling Framework) Ecore files

To start generating data, you need to open a valid EMF Metamodel Ecore file. Click on the Load Ecore Icon in the icon bar or choose File – Load Model from the menu bar. You can find predefined metamodel files in the artefacts subdirectory. Select your *.ecore file and click open.

 

Choose Artefacts

After you have selected a Ecore file, the “Choose Artefacts” window pops up. You can select an artefact from the Artefacts available box, and click the add button. Then, the artefact you have chosen will be available in the Artefacts selected box. When you mark the item, you will be able to select an Artefact type. There are already two artefacts predefined, Documents and Products, which can be selected by clicking the arrow of the drop-down box.

New Artefact Type

 If you need to specify a new or your own Artefact type, you can easily add and of course, remove artefact types. To do this, use the add button in the Artefact type area. Enter a name for your new artefact type and click OK. A check box gives you the opportunity to create physical dummy files to this kind of artefact type. But keep in mind, it is not always useful to do this!
