Setup Quantity / Generate Data

 

 

To start the data generating process, specify the amount that you want to generate. You can only generate data for specific attributes, yet. Enter a integer value in the textbox and press Return to accept the value that you have entered. If you don't press Return, you won't get valid output.

 

 

Finally, click the generate  button in the icon bar or go to Run - Generate Data in the menu bar.

You will be asked where you want to store the generated files and the data generation process will start.
