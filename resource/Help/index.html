Introduction

DataGenerator is a Java Desktop application based on Swing Application Framework and can run on various operating systems.

Its purpose is to generate test data and has been developed for the Chair of Media Informatics at the University of Bamberg within the scope of a Bachelor Thesis.

 

The actual version is 0.1
