System Requirements

 

It is recommend to install Java Runtime Environment 6 Update 12 or newer. You can download the latest version from http://www.java.com.

The program was tested on Windows (Windows Vista SP1, Windows XP Pro SP3) and Linux (openSUSE 11) Platforms.

In order to run the program, the computer must meet the minimum hardware requirements for Java 6.  This can be seen in the following table.
