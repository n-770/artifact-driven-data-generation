Distrbution

 

Before you can generate data, you have to set the distribution for each attribute you specified a filltype for. Click in the distribution row of each attribute. The program will show you the Distribution tab, where you can choose between the normal distribution and the uniform distribution method.

 

Uniform Distribution

Use the uniform Distribution when you want that all values will be used with the same probability. You can set the minimum (a) and the maximum (b) values. You can see in the pivot how many values are available.

 

 

Normal Distribution

The Normal Distribution is suitable for continual values and represents an ideal model. In the chart, you can see that there are 230 values availabe. Our configured the 75th as our expected value, but we also added a variance of 20, which means that we created a diversification of 20 values around our 75th value. In our example, there is now a diversification starting from 55 to 95. The more we move away from our expected value, the less the distribution that values will be used in this range.
